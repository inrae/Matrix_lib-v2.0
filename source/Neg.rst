Neg function
============

Genericity polymorphism 'neg' function

Returns
-------
	Matrix
	The opposite Matrix from self

Examples
--------
>>> m=rand(3)
>>> print(m)
| +5.000 | +0.000 | +7.000 |
| +9.000 | +4.000 | +8.000 |
| +9.000 | +4.000 | +6.000 |
printed
>>> print(-m)
| -5.000 | -0.000 | -7.000 |
| -9.000 | -4.000 | -8.000 |
| -9.000 | -4.000 | -6.000 |
printed
