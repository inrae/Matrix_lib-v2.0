Ne function
===========

Genericity polymorphism 'ne' function

=============== ================ =======================================
**Parameters**   *type*           **Description**
*mat*            Matrix           The matrix to test
=============== ================ =======================================

Examples
--------
>>> m=rand(3)
>>> n=rand(3)
>>> print(m==n)
False

Returns
-------
	bint
		Return the test of non-equality between the self matrix and the mat argument.
